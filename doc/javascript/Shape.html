Class: Shape

Shape(shape, id, probabilityWeight)

This class represents a shape

Constructor

new Shape(shape, id, probabilityWeight)

This class provides a complex shape based on an 1D array definition
Basic operation are provided by this class
- Left/Right rotation
- Moving to the left/right/down
- Access to the orientation and position
Parameters:
Name Type Default Description
shape Array.<Number> 1D Array of the shape
id Number ID of the shape
probabilityWeight Number 12 Weight how often the shape should be used
